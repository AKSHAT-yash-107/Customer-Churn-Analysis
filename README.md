# Customer Churn Analysis Project

## Objective
Analyze telecom customer data to identify churn patterns and estimate revenue loss.

## Dataset
Telco Customer Churn dataset containing 7,000 customers with features like tenure, contract type, monthly charges and churn status.

## Tools Used
- SQL (segmentation & churn calculation)
- Excel Dashboard (visual reporting)



## Key Insights
- Month-to-Month contracts show highest churn (~45%)
- New customers (0â€“6 months) are most likely to leave
- High monthly charge customers churn more
- High-risk segment: new month-to-month users

## Financial Impact
- Churn Rate: 31.5%
- Annual Revenue at Risk: $1.68M

## Business Recommendations
- Convert customers to yearly contracts
- Provide onboarding support in first 6 months
- Offer retention incentives for high-paying users
